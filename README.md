
Project folder for Team 09 in the course Advanced Deep Learning for Robotics at TUM in WS2025/26.

## Quick-start

1. Install "astral-uv" package manager for Python (https://docs.astral.sh/uv/getting-started/)
2. Clone repo
3. Run code with `uv run yourscript.py`
4. Sign up for Weight&Biases (free academic plan), and install to monitor training (https://docs.wandb.ai/models/quickstart#python)

To add dependencies: `uv add name-of-package`

To run with jupyter notebook: `uv run --with jupyter jupyter lab`
